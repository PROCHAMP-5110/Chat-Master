# Chat-Master

A modern, dark-themed real-time chat application featuring user authentication (Log In / Sign Up) and secure audio voice messaging.

## 🚀 Features
* 🔐 **Secure Authentication:** User registration and login screens powered safely by Firebase Auth.
* 🎤 **Voice Messages:** Record up to 10 seconds of high-quality audio and send it instantly to the room.
* 💬 **Real-time Sync:** Messages and voice notes synchronize instantly across all users using Firebase Realtime Database.
* 🎨 **Dark Mode UI:** A sleek, modern user interface designed for comfortable reading and clean aesthetics.

## 🛠️ Setup Instructions
1. Clone or download this repository.
2. Open the `index.html` file in any web browser.
3. Sign up for a new account or log in to start chatting!
